# Kali Nmap Scans Lab

This repository contains a step‑by‑step lab guide for building a Kali Linux environment and performing basic network scans with Nmap.

## Contents

- **kali_nmap_scans_lab.docx** – Word document with instructions for installing Kali Linux, updating the system, installing Nmap, and executing ping sweeps, TCP SYN scans, version detection, OS fingerprinting and saving results.

## Summary

The guide demonstrates how to:

1. Create a virtual machine, install Kali Linux and configure it.
2. Update packages and install Nmap.
3. Identify your local subnet and perform a host discovery scan.
4. Scan target hosts for open ports, services and operating system.
5. Save scan results to a file and interpret the findings.

See the Word document for detailed steps and command examples.
